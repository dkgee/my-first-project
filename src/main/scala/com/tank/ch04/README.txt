 4.1 类、字段和方法
	     Public 是Scala的默认访问级别
	     Scala里方法参数的一个重要特征是它们都是val，不是var。
	     通常我们定义副作用为能够改变方法之外的某种状态或执行I/O活动的方法。
      4.2 分号推断
	     Scala程序里，语句末尾的分号通常是可选的。愿意可以加，若一行里仅有一个语句也可以不加。不过，如果一行包含多条
	     语句时，分号则是必须的
      4.3 Singleton对象
	     当单例对象与某个类共享同一名称时，它就被称为是这个类的伴生对象。类和它的伴生对象必须定义在一个源文件里。
	     类就被称为是这个单例对象的伴生类。类和它的伴生对象可以互相访问其私有成员。
	     类和单例对象的差别是，单例对象不带参数，而类可以。
	     单例对象在第一次被访问的时候才会被初始化。
	     不与伴生类共享名称的单例对象被称为独立对象。
