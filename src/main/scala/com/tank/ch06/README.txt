       6.7 辅助构造器
 	     Scala主构造器之外的构造器被称为辅助构造器(auxiliary constructor)
 	     Scala的辅助构造器定义于def this(...)
 	     Scala里的每个辅助构造器的第一个动作都是调用同类的别的构造器
 	     每个Scala的构造器调用终将结束于对主构造器的调用，因此主构造器是类的唯一入口点。
 	     Scala的类里面只有主构造器可以调用超类的构造器。
