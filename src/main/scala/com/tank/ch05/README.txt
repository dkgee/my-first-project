 5.1 基本类型
 	     整数类型：Byte、Short、Int、Long、Char
 	     数类型 = 整数类型 + Float + Double
 	     Scala社区实践提出的推荐风格是一直使用大写形式
       5.2 字面量
              字面量就是直接写在代码里的常量值。
 	     不论用什么形式的整数字面量初始化，Scala的shell始终打印输出基于10的整数值。
 	     如果数开始于零，就是八进制的，并且只可以包含数字0到7。
       5.3 操作符和方法
 	     任何方法都可以是操作符，到底是方法还是操作符取决于你如何使用它。如果写成s.indexOf('o'),
 	     indexOf就不是操作符。但如果写成 s indexOf 'o'，那么indexOf就是操作符。
 	     中缀操作符，前缀标注，后缀标注。
 	     标识符中，能作为前缀操作符的只有 +、-、！和~。
 	     惯例是如果方法带有副作用就加上括号。如果没有就去掉括号。
       5.7 对象的对等性
 	     Scala没有操作符，实际上，操作符只是方法的一种表达方式。
       5.8 操作符的优先级和关联性
 	     不论操作符具有什么样的关联性，它的操作数总是从左到右评估的。
 	     如果方法结束于:，它们就被自右向左分组；反过来，就是自左向右分组。
